# 🚗 Car Price Prediction Web Application

A machine learning web application that predicts used car prices based on brand, year of manufacture, and mileage. Built with Python, Scikit-learn, and Flask.

---

## ✨ Features

- **Price prediction** — estimates car price in both USD and PKR
- **Random Forest model** — better accuracy than Linear Regression (R²: 0.715, MAE: ~$4,955)
- **Data exploration notebook** — full EDA with visualizations and model comparison
- **Clean web interface** — dark modern UI with async prediction (no page reload)
- **Input validation** — handles invalid inputs gracefully with user-friendly error messages

---

## 🛠️ Tech Stack

- **Python** — core logic
- **Scikit-learn** — Random Forest, OneHotEncoder, Pipeline
- **Pandas / NumPy** — data cleaning and preprocessing
- **Matplotlib / Seaborn** — data visualization (notebook)
- **Flask** — web framework and REST API
- **HTML / CSS / JavaScript** — frontend interface

## 🚀 Setup & Run

### 1. Clone the repository
```bash
git clone https://github.com/your-username/car-price-prediction.git
cd car-price-prediction
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Train and save the model
```bash
python train_model.py
```
This generates `model.pkl` and `meta.pkl` in the project folder.

### 4. Run the app
```bash
python app.py
```

### 5. Open in browser
Visit: **http://127.0.0.1:5000**

> ⚠️ Always run scripts from inside the project folder, not from a parent directory.

## 📊 Model Comparison

| Model | R² Score | MAE (USD) |
|-------|----------|-----------|
| Linear Regression | 0.693 | $5,408 |
| Gradient Boosting | 0.699 | $5,421 |
| **Random Forest** | **0.715** | **$4,955** |

Random Forest was selected as the final model due to its highest R² and lowest error.

---

## 📋 How It Works

1. User selects car brand, enters year and mileage (km)
2. Mileage is converted from km to miles internally
3. The trained pipeline preprocesses inputs using OneHotEncoding for brand
4. A log-scale prediction is made and converted back to USD
5. Price is displayed in both USD and PKR (rate: 1 USD = 278 PKR)
